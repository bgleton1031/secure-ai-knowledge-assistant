# Secure AI Knowledge Assistant

## Project Summary
This project is a secure AI assistant that answers questions using only approved internal documents. It demonstrates foundational skills in AI, cybersecurity, access control, logging, and technical documentation.

## Goals
- Build a simple AI knowledge assistant
- Restrict answers to trusted documents
- Add basic security controls
- Log user activity
- Document the full build process

## Problem
Many AI systems generate responses from general knowledge, which can lead to inaccurate or unsafe answers. Organizations need AI systems that operate only on trusted data.

## Solution
This project uses a retrieval-based AI assistant that searches approved documents and generates responses based only on that content. Basic security controls are included to reduce risk.
