# Role-Based Access Control

## Roles
- user
- admin

## Access Rules
### user
Can access:
- password_policy.txt
- acceptable_use_policy.txt

Cannot access:
- incident_response.txt

### admin
Can access:
- password_policy.txt
- acceptable_use_policy.txt
- incident_response.txt

## Why This Matters
Role-based access control limits which documents a user can query.
This demonstrates a basic access restriction model for secure AI systems.
